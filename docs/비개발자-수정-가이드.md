
코드(`streamlit_app.py:275`)가 이 콤마를 제거하지 않고 숫자 변환을 시도했는데, pandas 2.2에서 기존 옵션이 제거되어 에러가 발생합니다.


"백업 완료" 응답을 받은 후에만 다음 단계로 진행하세요.

**어떻게 고치나**


**백업 먼저 (수정 6 필수)**

Claude Code에:

```
streamlit_app.py의 preprocess 함수(271-285행 부근)에 OD 데이터용 ZONE 조인 로직을 추가해주세요.

요구사항:
1. ORGN 컬럼이 있을 때만 동작 (사회경제지표는 건너뛰기)

3. 반드시 try/except로 감싸기. 실패 시 st.warning 띄우고 SIDO/SIGU 없이 계속 진행
4. 머지 후 기존 SIDO/SIGU 필터링 로직(276-279행)이 정상 동작하는지 확인

참고:
- load_integrated() 함수(326행)가 preprocess()를 호출하는 진입점
- get_sigu_list() 함수(138행)가 ZONE 탭 로드 예시
- .omc/plans/diagnose-and-fix-sheets-connectivity.md의 Option 2 설계 참조
```

**검증**

1. Streamlit 재시작 (또는 클라우드 push)
