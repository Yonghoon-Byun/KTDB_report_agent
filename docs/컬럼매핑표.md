# 원본 KTDB 컬럼 ↔ DB 스키마 매핑 표

**문서 ID**: MAP-2026-05-06-KTDB-COLUMNS
**작성일**: 2026-05-06
**대상 독자**: 데이터팀, 인계 부서, AI 라우팅 함수 개발자
**짝 SQL**: `db_schema_smr.sql` (모든 매핑이 `COMMENT ON COLUMN`으로도 보존됨)

## 1. 변환 정책

| 원칙 | 적용 |
|---|---|
| 컬럼명 | **영문 snake_case** (Postgres 표준 + AI route 함수 코드 단순화) |
| 한글 의미 | **DB COMMENT + 본 문서 양쪽**에 보존 (`\d+ smr_*`로 즉시 확인 가능) |
| 데이터 형식 | wide format → **long format** (인구·취업자만 해당, 자연어 질의 호환) |
| 1과 2 사이의 진실원 | **DB COMMENT** (코드와 동기화). 본 문서는 가독성용 미러 |

> ⚠ COMMENT와 본 문서가 어긋나면 DB COMMENT가 정답. 본 문서는 정기 갱신.

## 2. 존체계 (`smr_zones`)

원본: `존체계.xlsx` (1310행)

| 원본 컬럼 (한글) | DB 컬럼 | 타입 | 의미 |
|---|---|---|---|
| 시도 | `sido` | TEXT | 17개 시도 (서울/경기/인천 + 비수도권 14개) |
| 시군구 | `sigu` | TEXT | 시군구 |

## 3. 사회경제지표 4종

원본: `3. 사회경제지표/{인구수,종사자수,취업자수,학생수}/`

### 3.1 인구수 (`smr_socio_pop`)

원본 파일: `SUB_POPYY.TXT` (cp949, 공백 구분, 1138행 = 헤더 1 + 1137 권역내부)

원본 헤더: `일련번호 ZONE_ID 남0~100세 여 0~100세`

원본 wide 204컬럼 → long 5컬럼:

| 원본 wide 컬럼 | DB long 컬럼 | 의미 |
|---|---|---|
| (파일명에서 추출) | `year` | 2023, 2025, 2030, 2035, 2040, 2045, 2050 |
| `일련번호` | (저장 안 함) | zones FK로 매핑 검증만 |
| `ZONE_ID` | `taz_seq` | zones FK |
| `남0세` ~ `남100세` (101개) | `gender='M'`, `age=0..100` | 성별/연령 차원 |
| `여0세` ~ `여100세` (101개) | `gender='F'`, `age=0..100` | 동상 |
| 셀 값 | `value` (REAL) | 인구 수 (명, 집체사가구 제외) |

### 3.2 취업자수 (`smr_socio_emp`)

원본 파일: `EMP_POP_YY.TXT` (구조는 인구수와 동일 wide 204컬럼)

매핑은 `smr_socio_pop`과 동일 — long 변환 후 `value` = 취업자 수 (명).

### 3.3 종사자수 (`smr_socio_work`)

원본 파일: `WORK_POPYY.TXT` (헤더 4컬럼 = `일련번호 ZONE_ID 3차 총종사자`)

| 원본 컬럼 | DB long 컬럼 | 의미 |
|---|---|---|
| `3차` | `indicator='3RD '` | 3차 산업 종사자 |
| `총종사자` | `indicator='TOT '` | 전체 종사자 (전 산업) |

### 3.4 학생수 (`smr_socio_stu`)

원본 파일: `STU_POPYY.TXT` (헤더 7컬럼 = `일련번호 ZONE_ID 초 중 고 특수 대`)

| 원본 컬럼 | DB long 컬럼 | 의미 |
|---|---|---|
| `초` | `level='ELEM'` | 초등학생 |
| `중` | `level='MID '` | 중학생 |
| `고` | `level='HIGH'` | 고등학생 |
| `특수` | `level='SPEC'` | 특수학교 학생 |
| `대` | `level='UNIV'` | 대학생 |

## 4. 통행량 OD 3종

원본: `4. OD/` (utf-8/ASCII, 공백 구분, 헤더 없음, 1310² = 1,716,100 행/연)

### 4.1 목적OD (`smr_od_purpose`)

원본 파일: `ODTRIPYY_F.OUT` (9컬럼)

| 원본 위치 | 원본 의미 (한글) | DB 컬럼 | 비고 |
|---|---|---|---|
| col 1 | 출발존(TAZ) | `orgn_seq` | 1~1310 |
| col 2 | 출발존(행정기관코드) | (저장 안 함) | zones JOIN으로 lookup |
| col 3 | 도착존(TAZ) | `dest_seq` | 1~1310 |
| col 4 | 도착존(행정기관코드) | (저장 안 함) | 동상 |
| col 5 | 귀가 | `home` | |
| col 6 | 출근 | `work` | |
| col 7 | 등교 | `scho` | |
| col 8 | 업무 | `busi` | |
| col 9 | 기타 | `othe` | |

### 4.2 주수단OD (`smr_od_main_mode`)

원본 파일: `OD_MMODE_YY_F.TXT` (14컬럼)

| 원본 위치 | 원본 의미 | DB 컬럼 |
|---|---|---|
| col 1~4 | 4 ID (목적OD와 동일) | `orgn_seq, dest_seq` (admin_code 미저장) |
| col 5 | 도보/자전거 | `walk_bike` |
| col 6 | 화물/기타 | `freight` |
| col 7 | 기타버스 (시외/고속버스 등) | `etc_bus` |
| col 8 | 일반철도 | `rail` |
| col 9 | KTX | `ktx` |
| col 10 | 승용차 | `auto` |
| col 11 | 택시 | `taxi` |
| col 12 | 버스 | `bus` |
| col 13 | 지하철 | `subw` |
| col 14 | 버스+지하철 (연계) | `bus_subw` |

### 4.3 목적별주수단OD (`smr_od_purpose_mode`)

원본 파일: `OD_MMODE_<HOME|WORK|SCHO|OTHE>_YY_F.TXT` (14컬럼, 4목적 × 7년 = 28파일)

⚠ **BUSI(업무) 목적은 원본에 없음** — 목적OD는 5목적이지만 목적별주수단은 4목적뿐.

| 원본 위치 | DB 컬럼 |
|---|---|
| (파일명 prefix) | `purpose` ('HOME' / 'WORK' / 'SCHO' / 'OTHE') |
| col 1~4 | `orgn_seq, dest_seq` |
| col 5~14 | `walk_bike, freight, etc_bus, rail, ktx, auto, taxi, bus, subw, bus_subw` (주수단OD와 동일 매핑) |

## 6. Sparse-drop 정책 (OD 전용)

모든 metric 컬럼이 0인 행은 **적재 단계에서 제거**. 합계는 보존(검증으로 확인).

| 데이터셋 | 2023 nonzero ratio | 7년 적재 추정 |
|---|---|---|
| 목적OD | 90.4% | ~10.85M 행 |
| 주수단OD | 90.4% | ~10.85M 행 |
| 목적별주수단OD HOME | 76.3% | |
| 목적별주수단OD WORK | 63.7% | |
| 목적별주수단OD SCHO | **19.3%** | (가장 sparse) |
| 목적별주수단OD OTHE | 80.9% | |
| 목적별주수단OD 합계 | 평균 60% | ~28.84M 행 |

## 7. AI route 함수와의 매핑

`streamlit_app.py`의 `ai_route()` 함수가 자연어 질의를 dataset+컬럼으로 라우팅할 때 본 매핑 표를 참조.

### 7.1 한글 키워드 → DB 컬럼 매핑 (예시)

| 사용자 한글 키워드 | dataset | 컬럼 |
|---|---|---|
| 인구, 인구수 | smr_socio_pop | value (집계 SUM) |
| 30대 여성 인구 | smr_socio_pop | value WHERE gender='F' AND age BETWEEN 30 AND 39 |
| 종사자, 일자리 | smr_socio_work | value WHERE indicator='TOT ' |
| 3차산업 | smr_socio_work | value WHERE indicator='3RD ' |
| 취업자 | smr_socio_emp | value |
| 초등학생 | smr_socio_stu | value WHERE level='ELEM' |
| 출근 통행 | smr_od_purpose | work |
| 등교 통행 | smr_od_purpose | scho |
| 승용차 통행 | smr_od_main_mode | auto |
| KTX | smr_od_main_mode | ktx |
| 출근 + 승용차 | smr_od_purpose_mode | auto WHERE purpose='WORK' |
